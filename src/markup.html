How to: Create Quick Steps in Outlook 2010

Solution

If you created personal folders (PSTs) in Microsoft Outlook previously and the PSTs have not yet been locked down, you can still save files/folders to them to reduce the size of your mailbox. Follow the best practices listed below to prevent Outlook perfermance issues.

Best Practices for PSTs

  • Each PST should be no larger than 4GB
  • No more than 3 PSTs should be mapped to your mailbox at any given time
  • A large personal folder can become corrupted and result in data loss. While the size at which a personal folder may corrupt is not definitive, as a general guideline keep the PST (including subfolders) to 4GB. If you already have a larger PST in place and are not experiencing any problems, just start using a new one
  • Use PSTs to store data that is not frequently accessed and used day to day. Leave frequently accessed content in your e-mail folders

Unmap/Disconnect a PST Folder Right-click against the top-level PST in the Outlook navigation pane and select Close "Personal Folder"

Re-map/Connect a PST Folder
